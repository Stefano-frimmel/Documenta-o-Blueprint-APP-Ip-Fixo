## Requisitos

O sistema permite configurar redes IP, PPPoE e DHCP e gerar relatórios. Prioriza simplicidade e segurança.

### Requisitos Funcionais
- RF-01: Autenticação Firebase
- RF-02: IP Fixo (IP, Máscara, Gateway)
- RF-03: PPPoE (Login, Senha)
- RF-04: DHCP automático
- RF-05: Relatórios PDF/CSV
